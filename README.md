**House Prices: Advanced Regression Techniques**
Реализация решения для популярного соревнования Kaggle по прогнозированию стоимости недвижимости в Эймсе, штат Айова.

Текущий результат: 
  **RMSE (log): 0.12721**
Место в лидерборде: 
  **~1440 (Top 10-14%)**


**Цель проекта** —
Предсказать цену продажи каждого дома (SalePrice)
На основе 79 категориальных и вещественных признаков,
Описывающих практически все аспекты жилых домов.

В данном решении реализован полный цикл разработки: от глубокого исследовательского анализа данных (EDA) до построения ансамбля моделей через Stacking.


**Архитектура решения**
Для начала проект был полностью реализован в Jupyter Notebook, но в последующем его реализация приняла следующий вид.
Проект разделен на модули для удобства поддержки и масштабируемости:
*database.py*: Загрузка данных и обработка кодировок.
*preprocessing.py*: Очистка данных, удаление признаков с пропусками >50%, обработка выбросов и трансформация признаков.
*models.py*: Архитектура ансамбля (StackingRegressor), объединяющая 8 различных моделей.
*visualization.py*: Инструменты для анализа распределения цен и важности признаков.
*main.py*: Точка входа в программу.


**Методология (Pipeline)**
1. *Обработка данных*
Логарифмирование: Применение np.log1p к целевой переменной для нормализации распределения.
Импутация: Заполнение пропусков медианой для чисел и модой для категорий.
Кодирование: Использование One-Hot Encoding для категориальных признаков.

2. *Моделирование*
Использован StackingRegressor, объединяющий сильные стороны различных алгоритмов:
Линейные: Ridge, Lasso, ElasticNet
Деревья: RandomForest, ExtraTrees
Бустинги: XGBoost, LightGBM
Мета-модель: Ridge (финальный агрегатор)


**Результаты**
Модель показывает стабильные результаты благодаря усреднению предсказаний разных типов моделей.
LightGBM (RMSE(log) - 0.1312, MAE - ~16,2)
Stacking (RMSE(log) - 0,1272, MAE - ~15,1)
(Использовали для наглядного выявления эффективности ансамблевого подхода к обучению)

**Вывод**
По итогу в проекте представлен полный Pipeline от Разведочного анализа данных (EDA) до обучения моделей и визуализации получившихся данных.
